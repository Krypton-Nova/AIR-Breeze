[Enter] = Confirm dialogue (useful in WA)
[Backspace] = Previous page
[Spacebar] = Reports page
[Ctrl] + [Tab] = Switch browser tab
[Ctrl] + [W] = Close browser tab

[F5] = Refresh page
[F6] = Select link to current page. Press [Ctrl] + [C] to copy
[F7] = Previous page
[F8] = Next page
WARNING: Reload page after using [F7], [F8] or [Backspace]!

[1-5] = Open nation on left side to doss
Keypad [1-5] = Same as above but on the right side

[A] = Activity (spotting). Press 2nd time to remove filters
[B] = The Pareven Isles
[C] = Cross (open nations endorsing this nation)
[D] = Doss (nation or region)
[E] = Endorse
[J] = Apply/Join/Leave WA
[L] = Apply/Join/Leave WA
[M] = Move to region
[N] = Nation
[R] = Region
[U] = Check if you updated
[W] = Open region, press again to view WAD
[X] = Open dossier, press again to clear
